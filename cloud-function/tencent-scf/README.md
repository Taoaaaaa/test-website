# 腾讯云SCF云函数部署指南

## 免费额度

- 每月100万次调用
- 每月40万GBs执行时间
- 最长900秒超时

## 部署步骤

### 1. 注册腾讯云账号

访问 https://cloud.tencent.com 注册并登录

### 2. 开通云函数SCF

进入 https://console.cloud.tencent.com/scf 开通服务

### 3. 创建云函数

1. 点击「新建」→「从头开始」
2. 配置：
   - 函数类型：事件函数
   - 函数名称：smarttable
   - 运行环境：Node.js 16.13
   - 执行方法：index.main_handler
   - 执行超时时间：**300秒**
   - 内存：256MB

### 4. 上传代码

将 `index.js` 内容复制到代码编辑器中

### 5. 配置环境变量

在「函数配置」→「环境变量」中添加：

```
ZHIPU_API_KEY = 你的智谱API Key
```

### 6. 创建触发器

1. 点击「触发管理」→「创建触发器」
2. 配置：
   - 触发方式：API网关触发器
   - 请求方法：POST
   - 发布环境：发布
   - 勾选「启用集成响应」

### 7. 获取访问URL

创建成功后，复制API网关访问地址，格式类似：
```
https://service-xxx.gz.apigw.tencentcs.com/release/smarttable
```

### 8. 配置网站

在网站API配置中：
- 调用模式：选择「云函数模式」
- 云函数URL：粘贴上面的访问URL

## 注意事项

1. 超时时间建议设置300秒，足够应对API高峰期
2. API Key通过环境变量配置，不要硬编码在代码中
3. 免费额度足够个人使用
